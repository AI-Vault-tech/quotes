# Veritas - Where Wisdom Finds Its Voice

A digital sanctuary for wisdom, a library of human thought, and an interactive experience that makes timeless wisdom accessible, beautiful, and personally transformative.

## Project Overview

Veritas is not just another quote website, but a carefully crafted digital library that curates the most profound thoughts across 5,000 years of human history. Built with Next.js, TypeScript, and Tailwind CSS, Veritas offers:

- **Academic Rigor**: Every quote verified with primary sources
- **Personal Transformation**: Meaningful learning journeys that connect wisdom to personal growth
- **Beautiful Design**: Immersive experiences that honor the profundity of human thought
- **Living Library**: A growing collection that evolves with new insights and community contributions

## Features Implemented

### Core Architecture
- [x] Home Page: "The Wisdom Gateway" with dynamic quote display
- [x] Explore Wisdom: Multi-dimensional navigation by era, theme, school, and personality
- [x] Authors Library: Interactive profiles with biographies and influence networks
- [x] Collections: Curated learning paths and wisdom journeys
- [x] Personal Sanctuary: User dashboard with saved quotes, journal, and progress tracking

### Premium Features
- [x] Wisdom Match AI: Personality assessment matching users with compatible philosophers
- [x] Context Explorer: Historical and philosophical context for quotes
- [x] Quote Anatomy: Interactive breakdown of rhetorical devices and philosophical concepts
- [x] Wisdom Podcast: Daily audio readings and thematic series

### Technical Excellence
- [x] Performance Optimization: Text-first loading, progressive image loading, intelligent caching
- [x] Advanced Search: Semantic search by meaning, mood-based filtering, complexity adjustment
- [x] Responsive Design: Mobile-first approach with adaptive layouts
- [x] Accessibility: WCAG 2.1 AA compliance considerations

## Technology Stack

- **Framework**: Next.js 13+ with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: Custom-built with accessibility in mind
- **Data**: In-memory data structures (scalable to database integration)
- **Deployment**: Vercel-ready with optimization

## Project Structure

```
src/
├── app/                 # Next.js app router pages
│   ├── explore/         # Explore wisdom by various dimensions
│   ├── authors/         # Author profiles and library
│   ├── collections/     # Curated learning paths
│   ├── my-sanctuary/    # User personalization dashboard
│   ├── quotes/          # Individual quote pages
│   ├── api/             # API routes for data access
│   └── ...              # Other pages
├── components/          # Reusable UI components
├── data/                # Content data and type definitions
├── lib/                 # Utility functions and helpers
└── styles/              # Global styles and theme definitions
```

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run development server**:
   ```bash
   npm run dev
   ```

3. **Build for production**:
   ```bash
   npm run build
   ```

4. **Start production server**:
   ```bash
   npm start
   ```
